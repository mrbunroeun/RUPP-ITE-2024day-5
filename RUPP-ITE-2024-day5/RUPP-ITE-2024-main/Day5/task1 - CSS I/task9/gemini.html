Alignment Techniques Text Alignment: text-align: Used to align text within an element. left: Aligns text to the left. center: Centers text within the element. right: Aligns text to the right. justify: Justifies text, aligning both the left and right margins. Vertical Alignment: vertical-align: Used to align elements vertically within their parent container. top: Aligns the top of the element with the top of the container. middle: Centers the element vertically within the container. bottom: Aligns the bottom of the element with the bottom of the container. baseline: Aligns the baseline of the element with the baseline of the parent container. Display Properties block: Elements are displayed as blocks, taking up the full width of their parent container and starting on a new line. inline: Elements are displayed inline, without starting on a new line. inline-block: Elements are displayed as blocks, but without starting on a new line. flex: Elements are displayed as flex items, allowing for flexible layout and alignment.
